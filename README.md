# Realtime Messenger

Учебный мессенджер на Node.js, Express и Socket.IO.

## Что умеет

- регистрация и вход по номеру телефона и паролю
- личные переписки между конкретными аккаунтами
- сообщения сохраняются на сервере
- синхронизация в реальном времени
- статус «в сети» для активных пользователей
- поиск пользователей по номеру телефона
- индикатор доставки сообщения: одна маленькая точка
- индикатор прочтения сообщения: две маленькие точки
- настройки аккаунта:
  - изменение имени
  - изменение фото профиля

## Установка

```bash
npm install
npm start
```

После запуска открой:

```bash
http://localhost:3000
```

## Как проверить

1. Открой приложение в одном окне браузера.
2. Зарегистрируй первый аккаунт.
3. Открой приложение во втором окне или в режиме инкогнито.
4. Зарегистрируй второй аккаунт.
5. Найди собеседника по номеру телефона или выбери его в списке.
6. Отправь сообщение.
7. Одна точка означает доставку, две точки — прочтение.
8. В настройках можно изменить имя и фото профиля.

## Где хранятся данные

- `data/users.json` — аккаунты
- `data/messages.json` — личные сообщения
- `public/uploads` — фото профиля

## Ограничения

- пароли пока хранятся без хеширования
- нет JWT/сессий
